Descomprime el archivo en una carpeta donde tengas acceso, por ejemplo: a

C:\Users\TuUsuario\Herramientas\jdk-17

Cuando lo tengas descomprimido:

    Abre la terminal de Windows (cmd o PowerShell).
    Ve a la carpeta bin del JDK, por ejemplo:
        cd C:\Users\TuUsuario\Herramientas\jdk-17\bin
    Ejecuta:
        java -version
    Y:
        javac -version

    Si te muestra la versión de Java, ya está funcionando
